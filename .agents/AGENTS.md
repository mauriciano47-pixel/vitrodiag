# Reglas del Workspace — VitroDiag

- **Corrección Proactiva de Lints y Advertencias**: Cada vez que se realicen cambios, auditorías o se generen archivos (código Python, JavaScript o Markdown) y el IDE/terminal reporte advertencias o problemas (como errores de formato Markdown `MD022`, `MD032`, `MD009`, o advertencias de sintaxis y estilo de código), el agente debe proceder de forma autónoma a corregirlos de inmediato para mantener el workspace libre de alertas, sin esperar a que el usuario lo delegue explícitamente y sin necesidad de pedir autorización o confirmación previa.
