# VitroDiag v1.0.47

**Muestreo Rápido NNPB para Operador**

## Descripción
VitroDiag es una aplicación web para análisis de botellas de vidrio en tiempo real usando la cámara del celular.

## Características
- 📱 Acceso a cámara trasera del smartphone
- 🤖 Motor de inferencia analítica (Cerebro NNPB)
- 🎯 Detección de defectos en envases
- ⚡ Procesamiento en tiempo real

## Cómo usar
1. Abre la app en tu celular desde: https://mauriciano47-pixel.github.io/vitrodiag/
2. Permite acceso a la cámara
3. Alinea el envase según las guías
4. Haz clic en "DIAGNOSTICAR ENVASE"

## Defectos detectados
- 🚨 REBABA EN LA BOCA (OVERPRESSED)
- 🚨 BAJO BOCA / BOCA INCOMPLETA
- 🚨 COLUMPIO DE CUERPO (BIRDSWING)
- ✅ ENVASE CONFORME
